# AI Engineering

Applied LLM systems — retrieval, orchestration, and agent tooling — built to solve a specific business task, not just call an API and print the response.

## Focus

- Retrieval-Augmented Generation (RAG): chunking, embeddings, vector search
- LLM orchestration with LangChain/LlamaIndex
- Tool-calling agents
- Evaluation of LLM outputs (not just "it looks right")

## Why This Matters for Recruiters

Most student portfolios mention "AI" without a working system behind it. These projects ship a runnable backend (FastAPI) with a documented retrieval and evaluation strategy — the part that separates an AI engineer from someone who has only used ChatGPT.
