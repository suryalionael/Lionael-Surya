# CV Generator (Google Apps Script)

**Status:** ✅ Shipped — live repo: [github.com/suryalionael/cv-generator-gas](https://github.com/suryalionael/cv-generator-gas)

## Business Problem

PT Magna Solusi Indonesia's HR team manually formatted candidate CVs into a consistent company template for every new hire and applicant — a repetitive, error-prone task that ate into time better spent actually reviewing candidates.

## What It Does

A Google Apps Script tool that takes structured candidate input (via a Google Form or Sheet) and automatically generates a formatted, branded CV document — removing the manual copy-paste-reformat step entirely from the HR workflow.

## Tech Stack

- **Platform:** Google Apps Script
- **APIs:** Google Docs API, Google Sheets API
- **Language:** JavaScript (Apps Script runtime)

## Why It's Here

This is the proof-of-finishing project in this portfolio — built for and used by a real HR team, not a hypothetical user. It demonstrates the full loop: understand a workflow pain point, scope a small automation, ship it, and have someone other than me actually use it.
